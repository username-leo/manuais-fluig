# KANBAN de Movimentação

Processo desenvolvido no Fluig para cadastro de cartões KANBAN com impressão de etiqueta e QR Code, e movimentação de estoque entre armazéns via leitura do QR Code, integrado à API REST do Protheus (MATA261).
